🍪 起点Cookies提取工具

用于NothingBut Library应用的起点网站Cookies提取

📖 使用说明：
这个工具会从当前页面提取cookies。请按照以下步骤操作：
1
打开起点网站
在新标签页中访问 https://www.qidian.com
2
完成验证
如果出现滑动验证码或其他验证，请完成它
3
在起点页面打开Console
按 F12 打开开发者工具，切换到 Console 标签
4
运行提取脚本
复制下面灰色框中的JavaScript代码，粘贴到Console中，按回车
// 在起点网站的Console中运行此代码 (function() { const cookies = document.cookie.split(';').map(c => c.trim()).filter(c => c); if (cookies.length === 0) { console.error('❌ 未找到cookies'); return; } console.log('=== Cookies (配置文件格式) ==='); cookies.forEach(c => console.log(c)); console.log('\n=== Cookies (环境变量格式) ==='); console.log('export QIDIAN_COOKIES="' + cookies.join('; ') + '"'); })();
5
复制输出结果
在Console中会显示提取的cookies，复制它们保存到 qidian_cookies.txt 文件中
⚠️ 安全提示：
• 请勿分享你的cookies给他人
• Cookies包含登录信息，泄露可能导致账号风险
• 只保存在本地，不要上传到公共位置
💡 提示：
• Cookies会过期（通常几小时到几天），过期后需要重新获取
• 如果测试失败，可能是cookies过期或格式错误
• 保存格式：每行一个cookie，格式为 名称=值

📝 配置文件示例

_csrfToken=xxxxxxxxxxxxxxxxxx newstatisticUUID=xxxxxxxxxxxxxxxxxx _ga=GA1.2.xxxxxxxxxx.xxxxxxxxxx _gid=GA1.2.xxxxxxxxxx.xxxxxxxxxx

🚀 运行测试

cd tests/scraper-test # 方式1: 使用配置文件 # 将cookies保存到 qidian_cookies.txt cargo run --bin test_with_cookies # 方式2: 使用环境变量 export QIDIAN_COOKIES="你的cookies字符串" cargo run --bin test_with_cookies
