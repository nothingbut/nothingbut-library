## 🚀 测试执行步骤

### 准备阶段

1. **启动 Ollama**
   ```bash
   ollama serve
   ```

2. **下载模型**
   ```bash
   ollama pull qwen2.5:7b
   ollama pull nomic-embed-text
   ```

3. **准备测试数据**
   - 导入至少 2 本书
   - 每本书至少 10 个章节
   - 内容多样化（不同情节、人物）

4. **启动应用**
   ```bash
   bun run tauri:dev
   ```

### 执行阶段

1. **基础功能测试**（30 分钟）
   - 按照 TEST_STREAMING.md 测试对话
   - 按照 TEST_VECTOR_SEARCH.md 测试搜索

2. **性能测试**（20 分钟）
   - 测量响应时间
   - 测量搜索性能
   - 测量资源使用

3. **边界测试**（20 分钟）
   - 测试异常输入
   - 测试错误场景
   - 测试并发限制

4. **UI 测试**（15 分钟）
   - 测试所有交互
   - 检查视觉效果
   - 验证用户体验

### 报告阶段

1. 填写测试报告
2. 记录所有问题
3. 提交改进建议
4. 更新文档
